  • Fill in the following sheets in the downloaded template

    Overview
    • There are tips and info in each column header.
    • Blue columns are required.
    • Gray columns are filled in by excel formula (but you can optionally overrule them).
    • Filling out the sheets in the order below ensures drop-down lists are populated.
    • Regarding Treatments, Tissues, Compounds, and LC Protocols sheets
      • These sheets auto-populated from the database.
      • If you have novel entries for any of these sheets, you have the option to add them yourself or leave them for the curator do it.
      • Any new entries will be validated manually by a curator.

  • Study

    • Name your study.

  • Tracers

    • Add any missing tracers.  
      • If you are familiar with the Tracer Name format, you can enter just the name, and move on to the next sheet.
      • Alternatively, you can enter values in all the other columns and the tracer name column will be automatically filled via an excel formula.  
        • A tracer with 2 labels is defined on 2 rows (one for each labeled element).
        • Both rows in a 2-label tracer must have the same (arbitrary) "Tracer Row Group" number. Without this number, the excel formula will not populate the Tracer Name column.

  • Infusates

    • Feel free to remove or edit infusates that are not used in your study.
    • Add any missing infusates.  
      • Add by name only or...
      • Alternatively, you can enter values in all the other columns and the infusate name column will be automatically filled via an excel formula.  
        • An infusate with 2 tracers is defined on 2 rows.
          • Both rows in a 2-tracer infusate must have the same (arbitrary) "Infusate Row Group" number. Without this number, the excel formula will not populate the Infusate Name column.
          • Select the Tracer using the drop-downs. If your desired tracer is not there, enter it in the Tracers sheet (see step 2).
        • Note that Infusate names include each tracer's concentration in square brackets immediately after each tracer name. E.g. BCAAs (VLI) {valine-[13C5,15N1][20]; leucine-[13C6,15N1][24]; isoleucine-[13C6,15N1][12]}

  • Animals

    • Name each mouse.
    • Use drop-downs for the following columns. If data is missing, enter it in the corresponding sheet.
      • Infusate
      • Treatment
      • Study

  • Samples

    • Editing sample names  
      • Note that scan labels (like "_pos") will have been removed automatically.
      • Samples containing "blank" will not have been automatically added.
      • The samples sheet should not contain blanks, so remove any rows that were autofilled, and...
      • Don't forget to mark any blanks in the Peak Annotation Details sheet with "skip" in the Skip column.
    • Use drop-downs for the following columns. If data is missing, enter it in the corresponding sheet.
      • Animal
      • Tissue

  • Sequences

    • Create a row for each time you ran a mass spec instrument.
    • Note the MS Run Name is automatically created via excel formula. You will be selecting an MS Run for each peak annotation file in the next sheet.

  • Peak Annotation Files

    • Each peak annotation file and its file format will have been filled in automatically.
    • Select the MS Run associated with each file using the drop-downs in the Default Sequence column.

  • Peak Annotation Details

    • The purpose of this sheet is to associate the headers in each peak annotation file (and mzXML files) with TraceBase samples. They are usually identical and often 1:1, but can have scan labels and can be 1:many.
    • Optional columns do not need to be filled in unless they differ, or you encounter an error during validation.
    • If there were any blank samples you had to remove from the samples sheet, you will need to put "skip" in the Skip column for those blank samples.
    • The MS Run will default to the name associated with the Default Sequence for the peak annotation file in the Peak Annotation Files sheet.
