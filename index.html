TraceBase

TraceBase is a mass spectrometry data repository and analysis tool for isotopic tracing studies used to quantify in vivo metabolism. It is designed to be downloaded and installed for individual labs so that lab members can make their natural isotopic abundance corrected data (along with associated experimental metadata) available to the rest of the lab. It currently supports data imported from AccuCor, IsoCor, and Iso-AutoCor. It also loads RAW and mzXML files for raw data retrieval. TraceBase organizes this data and computes useful metrics such as enrichment, normalized labeling, and FCirc. TraceBase data is easy to browse, collate, compare, and download.
